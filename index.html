um iPadian para Android.

Olá! Aqui é o Lucas, criador do projeto. Este é um trabalho que desenvolvi para que vocês possam usar o iPadian original no Android. Nele, também é possível baixar e utilizar aplicativos direto na versão recriada do app que existe para iOS. Este projeto é desenvolvido por uma criança com o auxílio de Inteligência Artificial (Gemini). Como o projeto ainda está em desenvolvimento, erros e bugs podem acontecer. Se você encontrar algum problema, por favor, abra uma Issue na nossa página do GitHub para ajudar a melhorar o app. Agradeço muito a paciência e o apoio de todos!
